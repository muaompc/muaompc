*****************
What is muAO-MPC?
*****************

:math:`\muaompc` is a code and data generation tool for 
model predictive control.
:math:`\muaompc` is free software released under the terms
of the GNU general public license version 3 (GPLv3).

:math:`\muaompc` mainly consists of the ``muaompc`` Python package.
